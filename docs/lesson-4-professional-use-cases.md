# AI in the Workplace

## Lesson 4: Professional Use Cases

### Overview
AI adds value to many workplace tasks. Know where it helps and where to be careful.

---

### Where AI Typically Helps

**Drafting and editing**
- First drafts, email replies, document polish

**Research and analysis**
- Quick overviews, synthesizing information

**Meeting support**
- Agendas, summaries, follow-up notes

**Brainstorming**
- Generating ideas, exploring options

**Learning and skill building**
- Explanations, practice, feedback

**Process improvement**
- Analyzing workflows, suggesting changes

**Communication**
- Adjusting tone, improving clarity

### Where to Be Careful

- Tasks requiring confidential information
- Tasks needing real-time or specific data
- Tasks where errors have serious consequences
